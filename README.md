### FanLetter

<페이지소개>

- 좋아하는 아티스트에게 팬레터를 남길 수 있는 페이지

<주요기능>

- 로그인하여 페이지 접근
- 아티스트를 선택하여 팬레터 작성
- 작성한 팬레터 수정,삭제
- 프로필 사진, 닉네임 변경
